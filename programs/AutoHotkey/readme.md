# My AutoHotkey Settings

## Requirements

You must install AutoHotkey v2.x to use this scripts.

```console
winget install AutoHotkey.AutoHotkey
```

## Usage

1. Start AutoHotkey Dash.
2. Run "Compile".
3. Select your `.ahk` file, set desnination path, and press "Convert".
4. Run created `.exe` to apply keymapping.
