======
Zellij
======

:URL: https://zellij.dev/
:Layer: Application layer

役割と利用スタンス
==================

ローカルでのターミナル作業における、マルチ作業用。

v0.44でWindowsもサポートするようになったため、
基本的にターミナルマルチプレクサーの領分はこのソフトに任せる。
ターミナルソフトウェアが持つタブ機能との棲み分けは下記の通り。

================ ==========================
機能             立ち位置
ターミナルアプリ プロジェクトごとの窓を管理
Zellij           プロジェクト内の画面分割
================ ==========================

インストール
============

aquaのグローバル設定で定義。

環境変数
========

:SHELL:
  Zellij内のペインが最初に起動するシェルはこの変数を参照する。

  現時点ではメインのシェルにNushellを使い、必要に応じてZellijを起動する運用のため、
  Nushellが動作する際の設定時に変数書き換えを行う。

:ZELLIJ_CONFIG_DIR:
  Zellijの設定ファイルの場所。

  本来はNushellのようにシンボリックリンクで問題ないのだが、
  Windows版の挙動がちょっと変なので環境変数で明示的にdotfiles配下を参照させる。

  現時点ではメインのシェルにNushellを使い、必要に応じてZellijを起動する運用のため、
  Nushellが動作する際の設定内で代入する。
