=======
Nushell
=======

:URL: https://www.nushell.sh/
:Layer: Shell layer

役割と利用スタンス
==================

標準利用のシェル。

少なくとも、OSのデスクトップからターミナルを起動する時はこのシェルから開始する想定。

インストール
============

aquaのグローバル設定で定義。

基本的にはソフトウェアの設定を用いて起動時のシェルに指定するが、
WSLへ ``wsl`` コマンドで接続する際には Bash を中継する。 [#]_

.. [#] aquaのパスがユーザー領域で、WSLへのアクセス時に ``PATH`` への設定追加が少々面倒そうだったため。
